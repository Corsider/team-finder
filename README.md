# team-finder
Team Finder project server

Run ```init.bat``` to start docker and fill DB with testing data \
Put .env file in project root:

```
POSTGRES_USER =
POSTGRES_PASSWORD =
POSTGRES_DB = 
DB_PORT = 
DB_HOST = 
TOKEN_SECRET = 
TOKEN_TIMEOUT_HOUR = 
TIMEOUT = 5
SERVER_ADDRESS = 0.0.0.0:8080
```
